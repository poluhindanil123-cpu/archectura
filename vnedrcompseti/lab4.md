# Диагностика компьютера с помощью программ-диагностик

## Цель работы
Проводить диагностику компьютера с помощью программ-диагностик.

## Общие сведения о диагностическом ПО

Программы для диагностики компьютера позволяют:

- Проверить конфигурацию компьютера:
  - количество памяти и её использование;
  - типы установленных дисков;
  - другие параметры системы.

- Проверить работоспособность устройств компьютера (прежде всего жестких дисков).

- Выявить «намечающиеся» дефекты дисков, возникающие из-за износа магнитной поверхности, и предотвратить потерю данных.

- Определить несложные неисправности аппаратуры или ошибки в конфигурировании системных файлов.

## Объекты диагностики

Диагностика может выполняться как для системы в целом, так и для отдельных модулей:

| Категория | Компоненты |
|-----------|-----------|
| Системные компоненты | Системная плата, процессор |
| Память | Стандартная, дополнительная и расширенная память |
| Видеоподсистема | Видеоадаптер, графический ускоритель |
| Накопители | Жесткие диски, приводы флоппи-дисков, CD/DVD-приводы |
| Устройства ввода | Клавиатура, координатные устройства |
| Интерфейсы | Последовательные и параллельные порты, SCSI-интерфейс |
| Периферия | Модемы, принтеры, сетевые устройства |

## Программа PC Wizard

### Описание
**PC Wizard** — программа, предоставляющая обширную информацию обо всех установленных на компьютере компонентах.

### Возможности программы

**Информация об оборудовании:**
- Оперативная память;
- Материнская плата;
- Устройства хранения и записи информации;
- Видеоподсистема;
- Сетевые устройства;
- Модемы;
- Принтеры и другие периферийные устройства.

**Информация об операционной системе:**
- Версия системы;
- Установленные шрифты и библиотеки;
- Параметры WinSock;
- Активные процессы;
- Имеющиеся модули и сервисы.

**Тестирование производительности:**
- Быстродействие процессора;
- Скорость работы жестких дисков;
- Производительность CD/DVD-привода.

### Системные требования
- Поддержка 32-битных операционных систем;
- Поддержка 64-битных операционных систем.

## Заключение

Использование диагностических программ, таких как PC Wizard, позволяет получить полную картину состояния компьютера, своевременно выявить потенциальные проблемы и обеспечить стабильную работу системы. Регулярная диагностика рекомендуется как для профилактического обслуживания, так и для troubleshooting при возникновении неполадок.

## Контрольные вопросы

 ## 1 Причины возникновения проблем с совместимостью ПО
Различия в операционных системах (ОС): ПО, разработанное для старых версий (например, Windows 7), может не функционировать в новых (Windows 10/11) из-за изменения API.

## 2 Методы уменьшения проблем с совместимостью ПО
Использование режима совместимости (Windows): Запуск программы от имени администратора или в режиме эмуляции старых версий ОС.
Обновление ПО и драйверов: Установка последних версий приложений и драйверов оборудования, поддерживающих текущую ОС.
